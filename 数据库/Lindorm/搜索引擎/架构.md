云原生多模数据库 Lindorm搜索引擎是基于云原生多模数据库 Lindorm核心能力实现的高性能、低成本、稳定可靠的分布式搜索引擎。它兼容开源ElasticSearch标准接口，可无缝作为宽表引擎的索引存储，加速检索查询。搜索引擎适用于订单、账单、日志等主流业务场景，同时支持AI场景的向量检索，是构建大模型应用的基础服务组件。

## **功能特性**

| **核心能力**     | **说明**                                                     |
| ---------------- | ------------------------------------------------------------ |
| **生态兼容**     | 兼容开源Elasticsearch标准API接口，支持开源ELKB生态对接。     |
| **多模融合统一** | - 与Lindorm宽表引擎无缝融合，作为宽表的搜索索引，形成多模统一存储检索能力。搜索索引主要面向复杂的多维查询场景，能够覆盖分词、模糊查询、聚合分析、排序翻页等场景。 - 与向量引擎和AI引擎融合，为大模型RAG系统提供必需的全文-向量融合检索能力，提升大模型回答准确度。 |
| **云原生弹性**   | 采用存算分离的技术架构，支持计算和存储资源的按需弹性，shard迁移无需搬迁数据，秒级完成，实现快速高效扩缩容节点。 |
| **低成本**       | - 支持多种低成本的存储介质、冷热分离、高压缩比算法，存储成本相比自建低50%。 - 支持企业级TTL，可以在索引粒度上设置数据生命周期，回收过期数据，降低成本。 |

-   智能索引优化：通过SearchIndex技术自动匹配最优索引，显著提升查询效率；
