# 家庭财务健康度 5 指标体系（普通家庭版）

> 目标：
> **判断家庭在收入中断、支出波动、人生阶段变化时，是否会发生结构性下滑**

------

## 指标一：应急覆盖月数（Emergency Coverage Months）

### 📌 衡量什么？

家庭在**零收入情况下**，能靠现有应急金维持多久。

### 📐 定义公式：

> **应急覆盖月数 =
> （可用应急金 − 必须预留的年度刚性支出） ÷ 月必要支出**

### ✅ 判断标准：

| 覆盖月数 | 健康度   |
| -------- | -------- |
| < 3      | 高风险   |
| 3–6      | 勉强安全 |
| 6–9      | 健康     |
| 9–12     | 稳健     |
| ≥12      | 冗余偏高 |

### ⚠️ 常见误区：

- 把定期投资、理财算进应急金
- 用“当前工资”而不是“必要支出”来算

------

## 指标二：负债收入比（Debt-to-Income Ratio，DTI）

### 📌 衡量什么？

家庭每月收入中，有多少被**债务锁死**。

### 📐 定义公式：

> **负债收入比 =
> 每月固定债务支出 ÷ 家庭月税后收入**

（债务包括：房贷、车贷、消费贷、信用卡最低还款）

### ✅ 判断标准：

| 比例   | 健康度 |
| ------ | ------ |
| ≤30%   | 安全   |
| 30–40% | 可控   |
| 40–50% | 偏高   |
| >50%   | 高风险 |

### ⚠️ 常见误区：

- 只算房贷，不算其他负债
- 用税前收入掩盖风险

------

## 指标三：收入稳定指数（Income Stability Index）

### 📌 衡量什么？

家庭**收入可持续性**，而不是收入高低。

### 📐 简化评分法（不复杂也够用）：

| 情况                | 得分 |
| ------------------- | ---- |
| 双职工              | +2   |
| 单职工              | 0    |
| 体制内 / 强稳定行业 | +2   |
| 私企 / 自由职业     | 0    |
| 有第二收入来源      | +1   |
| 行业周期强波动      | −1   |

**得分区间：0–5**

### ✅ 判断参考：

- **0–1：低稳定**
- **2–3：中等**
- **4–5：高稳定**

### ⚠️ 常见误区：

- 用“当前工资高”替代“长期稳定”
- 忽视行业周期风险

------

## 指标四：保障完整度（Protection Coverage）

### 📌 衡量什么？

**重大风险是否会直接摧毁家庭资产**

### 📐 核心检查项：

- 医保是否连续
- 是否有医疗险
- 是否有重疾险（家庭主要收入者）
- 是否有定期寿险（有房贷 / 子女家庭）

### ✅ 判断标准（简化）：

| 状态               | 健康度 |
| ------------------ | ------ |
| 仅医保             | 脆弱   |
| 医疗险齐全         | 基础   |
| 医疗 + 重疾        | 健康   |
| 医疗 + 重疾 + 定寿 | 完整   |

### ⚠️ 常见误区：

- 保险买得多 ≠ 保障完整
- 给孩子买重疾，却不给家庭经济支柱买寿险

------

## 指标五：储蓄缓冲率（Savings Buffer Ratio）

### 📌 衡量什么？

家庭在**不影响生存前提下**，还能主动调整的空间。

### 📐 定义公式：

> **储蓄缓冲率 =
> （家庭月结余） ÷ 家庭月收入**

### ✅ 判断标准：

| 比例   | 健康度 |
| ------ | ------ |
| <10%   | 紧绷   |
| 10–20% | 一般   |
| 20–30% | 健康   |
| >30%   | 弹性强 |

### ⚠️ 常见误区：

- 强行高储蓄导致生活质量塌陷
- 把投资收益当作稳定结余

------

## 六、如何综合看这 5 个指标？

### 👉 一个简单判断法：

- **至少 3 项达“健康”以上 → 家庭结构安全**
- **应急覆盖月数 + 负债收入比** 是底层安全线
- 其他 3 项决定“抗打程度”和“上限”

------

> **家庭财务健康，不是赚得多，而是遇到坏事不会倒。**
